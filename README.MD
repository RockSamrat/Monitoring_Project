# Flask + Prometheus + Grafana Monitoring Setup

A simple example demonstrating how to monitor a Flask application using Prometheus and Grafana with Docker Compose.

```bash

## Learnings
Through this project, I gained hands-on experience with application monitoring and observability using Flask, Prometheus, and Grafana. I learned how to containerize services with Docker Compose, expose 
application metrics, configure Prometheus for data collection, and create dashboards for visualizing system performance. Building the setup independently also improved my understanding of service 
orchestration, networking between containers, and the importance of monitoring and metrics in modern software deployment and maintenance.
